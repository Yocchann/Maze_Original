・迷路の難易度変更の仕方  
（初期設定は0.04で少し簡単にしています）。  
まず、ViewControllerを開きます。
```
playerMotionManager.accelerometerUpdateInterval
```
**このコードの = 先の値を変えることでプレイヤーの動きの速さを変えられます。**
